# Briefing Rules

## What to Include
- Urgent/critical items only
- Action items from meetings NOT already in the task list
- System alerts (disk anomalies, cost anomalies if tracked)

## What to Skip
- Marketing emails, newsletters, Substack updates
- "Notes received from X meeting" (unless actionable item missing)
- Calendar overview on weekends
- Routine family messages

## Task Review
- Only flag overdue/blocked items not caught by scheduled checks
- Do NOT duplicate what scheduled tasks already produced (briefing, checkpoint, email scan, meeting prep run automatically)
- Read the existing file before regenerating — if today's briefing already exists, use it
